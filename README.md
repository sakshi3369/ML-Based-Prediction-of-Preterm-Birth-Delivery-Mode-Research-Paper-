# ML-Based-Prediction-of-Preterm-Birth-Delivery-Mode-Research-Paper

Project Overview

This project develops machine learning models to predict preterm birth risk and mode of delivery (Normal vs C-section) using maternal health data. The objective is to enable early risk identification and support data-driven healthcare decision-making.

Key Highlights:

Built ML models including Logistic Regression and Random Forest
Achieved 100% accuracy (1.0) for preterm birth prediction using Logistic Regression
Achieved 92% accuracy (0.92) for mode of delivery prediction using Random Forest
Applied Stratified K-Fold Cross Validation for robust model evaluation
Evaluated models using Accuracy, Precision, Recall, F1-score, and ROC-AUC

Statistical Techniques:

Chi-square test to analyze association between BMI and preterm birth
Mann-Whitney U test to compare distributions across delivery groups
Exploratory data analysis and visualization to identify key trends

Key Insights:

Logistic Regression performed best for preterm birth prediction
Random Forest outperformed other models for mode of delivery prediction
Factors like BMI, age, and health indicators significantly impact outcomes

 Impact:
 
Enables early identification of high-risk pregnancies
Supports clinicians in making data-driven decisions
Demonstrates real-world application of ML and statistical analysis in healthcare
